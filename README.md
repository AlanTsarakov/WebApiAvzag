Бэкенд-сервис, реализующий REST API для мобильного или веб-приложения "Авзаг" — платформы для изучения осетинского языка с нуля с помощью игровых методик и коротких уроков.

ASP.NET Core API
Swagger/OpenAPI 3.0

API Endpoints
1. GET /api/FAQ
Возвращает часто задаваемые вопросы, сгруппированные по темам.
2. GET /api/Question
Получает список вопросов/упражнений для урока.
Параметры:
sectionId (обязательный) - ID раздела
chapter (обязательный) - номер главы
lesson (обязательный) - номер урока
